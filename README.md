# Sharvari Portfolio

A modern personal portfolio website for **Sharvari Karnase**, a **Full Stack Java Developer** focused on building secure, scalable, and user-friendly web applications using **Java, Spring Boot, React.js, and MySQL**.

This portfolio is designed to showcase professional skills, featured projects, internship experience, education, certifications, contact information, a downloadable resume, and a profile photo in a polished, recruiter-friendly layout.

## About The Project

This project was built to create a strong online presence and present technical work in a clean, responsive, and visually engaging format. It highlights both frontend presentation and backend-oriented development strengths.

The website includes:
- A hero section with introduction, profile photo, and resume download
- An about section with professional summary and strengths
- A skills section covering frontend, backend, database, tools, and concepts
- A projects section featuring real-world full-stack applications
- An experience section with internship details
- An education section
- A certifications section
- A contact section with direct links

## Tech Stack

- **Frontend:** React.js
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel

## Features

- Fully responsive layout for desktop and mobile
- Smooth and modern UI design
- Animated sections and transitions
- Downloadable resume from local assets
- Integrated profile picture from assets
- Project showcase with GitHub and live demo links
- Professional experience and certification display
- Contact shortcuts for email, GitHub, and LinkedIn
- Fast build and deployment workflow with Vercel
